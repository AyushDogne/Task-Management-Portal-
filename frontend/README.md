## How to run backend and frontend

```bash
# 1) Clone project
git clone <YOUR_GIT_REPO_URL>
cd "Task Management Portal"

# 2) Install & run backend
cd backend
npm install
copy .env.example .env  # on PowerShell / CMD
npm run dev

# 3) Install & run frontend (new terminal)
cd "..\frontend"
npm install
npm run dev
```
